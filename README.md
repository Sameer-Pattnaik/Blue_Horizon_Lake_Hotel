

# 🏨 Blue Horizon Lake Hotel – Hotel Booking System

A **full-stack hotel booking web application** that allows users to browse rooms, make reservations, and manage bookings.
The system also provides an **admin dashboard** to manage rooms and bookings.

This project demonstrates **full-stack development using React and Spring Boot** with REST APIs and a database backend.

---

# 🚀 Features

## 👤 User Features

* Browse available hotel rooms
* View room details with price per night
* Book rooms instantly
* Search bookings using booking confirmation
* View hotel services
* Responsive modern UI

## 🛠 Admin Features

* View all existing rooms
* Add new rooms
* Edit room details
* Delete rooms
* View all bookings
* Cancel bookings
* Filter bookings by date

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/hotel-booking-system.git
```

---

## 2️⃣ Backend Setup

Navigate to backend folder:

```bash
cd backend
```

Run Spring Boot application:

```bash
mvn spring-boot:run
```

Backend will run on:

```
http://localhost:9192
```

---

## 3️⃣ Frontend Setup

Navigate to frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```
